Privacy Policy

MoveChat — Chrome Extension | Last updated: June 2026

Overview

MoveChat is a Chrome extension that captures AI chat sessions from supported platforms (ChatGPT, Claude, Gemini, Perplexity) and allows users to transfer them to other platforms.

Data Collection

MoveChat does not collect any data.

  • No analytics or telemetry
  • No tracking of user behavior
  • No remote servers
  • No data is sent to any third party except the AI provider you explicitly configure

Data Storage

All data is stored locally on your device using Chrome's chrome.storage.local:

  • Conversation transcripts captured from chat platforms
  • Extension settings (theme, preferences, handoff templates)
  • API keys for AI compression (if you choose to configure them)

This data never leaves your device unless you explicitly export it or resume a conversation on another platform.

API Keys

If you enable AI compression, you provide an API key for OpenAI, Anthropic, or Google Gemini. This key:

  • Is stored locally in chrome.storage.local
  • Is sent only to the API endpoint of the provider you select
  • Is never transmitted to any other server
  • Can be deleted at any time via the extension's settings (Factory Reset)

Permissions

PermissionPurpose
activeTabCapture conversations from the currently active tab
storageStore conversations, settings, and API keys locally
tabsOpen new tabs when resuming conversations on other platforms
scriptingInject content scripts into chat platform pages
*://*/*Fetch images from third-party CDNs for PDF export

The broad *://*/* host permission exists solely because images in conversations may be hosted on CDNs outside the four main platform domains (e.g., OpenAI's DALL-E images on Azure Blob Storage). This permission is used only for fetching those images, not for accessing any other data.

Third-Party Services

MoveChat does not integrate with any third-party analytics, advertising, or tracking services.

The only external communication is between your browser and the AI provider API endpoint you explicitly configure (e.g., api.openai.com for OpenAI). This communication is initiated by you, not by the extension.
